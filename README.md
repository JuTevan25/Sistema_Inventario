📋 Sistema de control de inventario - Consola C#

Sistema funcional desarrollado en Visual Studio con C# para el registro de productos en una bodega.  
Diseñado para practicar y perfeccionar **validaciones, métodos, bucles** y reducir errores al ingresar datos.

🚀 Características

- ✅ Validación de PIN con acceso denegado tras 3 intentos.
- ✅ Registro de nombre del producto (no permite vacíos ni nombres con espacios).
- ✅ Ingreso de código numérico entre 1 y 99, con validación de rango.
- ✅ Registro de cantidad en stock.
- ✅ Posibilidad de actualizar el precio del producto.
- ✅ Impresión de reporte final con todos los productos registrados.
- ✅ Validación en cada entrada del usuario para evitar errores.

🛠️ Tecnologías

- **Lenguaje:** C#
- **Plataforma:** .NET (cualquier versión que soporte consola)
- **Compilador:** dotnet, Visual Studio o csc

📦 Requisitos previos

- [.NET SDK](https://dotnet.microsoft.com/download) (recomendado 6.0 o superior)
- [Visual Studio Community](https://visualstudio.microsoft.com/es/vs/community/)
- O Visual Studio Code con extensión C#
